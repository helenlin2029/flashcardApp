# Flashcard App
A web-based flashcard quiz app build with Python and Flask. 
Users can create decks, edit content (add/remove/modify cards) and test knowledge with a built-in quiz mode that uses spaced repetition, focusing on cards that the user finds more difficult, returning an accuracy score once a quiz is complete. 


# Features
- Create and delete decks
- Add, delete, and modify cards within decks
- Built-in quiz mode
- Space repetition: pushing cards that the user consistently get incorrect more frequently

# Tech Stack
Frontend: HTML/CSS/JS
Backend: Flask, Python
Storage: JSON

# How to Run
1.) Clone repository
2.) Install dependencies (requirements.txt)
